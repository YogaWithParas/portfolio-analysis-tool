## 🚀 **Installation & Setup**

### **Prerequisites**
```bash
# System Requirements
Python 3.8+ (Data Science Stack)
Git (Version Control)
Virtual Environment (Isolation)
```

### **Professional Setup**
```bash
# 1. Clone Repository
git clone https://github.com/YogaWithParas/portfolio-analysis-tool.git
cd portfolio-analysis-tool

# 2. Environment Setup
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 3. Install Dependencies
pip install -r requirements.txt

# 4. Launch Application
python enhanced_dashboard.py
```
